# One Sensor, One Year

A recurring data-art series. Each edition takes a single data stream, visualizes its entire year as generative art, and pairs it with a narrative essay that tells the human story the data contains.

The art makes you feel it. The story makes you understand it.

## About

Created by **Pushpesh Sharma** -- bridging technical depth with creative vision, turning infrastructure data into art and narrative.

Each edition follows the same format:
- **The Art** -- a single generative visualization encoding an entire year of data
- **The Anatomy** -- an annotated version highlighting key moments
- **The Story** -- a ~1,000-word narrative essay
- **The Interactive** -- a web page for exploring the data
